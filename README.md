# DayVo
A personal daily dashboard  weather, a live clock and calendar, a Pomodoro timer, a to-do list, an hour-by-hour daily planner, daily goals tracking, and motivational quotes, all in one responsive single-page app.  Built with vanilla HTML, CSS, and JavaScript with no frameworks, no build step.


Features


Live weather — search any city or use your current location. Background and icon change based on conditions (sunny, cloudy, rainy).
Live clock & calendar — current time updates every second, with the current month shown as a calendar and today's date circled.
Pomodoro timer — Focus / Short break / Long break modes, customizable durations, adjustable alarm volume and sound, with the timer synced between the dashboard card and the full timer page.
To-do list — add, edit, and delete tasks with priority and status. High-priority tasks surface on the dashboard card with a quick-check checkbox.
Daily planner — a 24-hour editable schedule (3 columns on desktop, 2 on tablet, 1 on mobile), with the current hour's task shown on the dashboard card.
Daily goals — a simple editable checklist with live progress percentage on the dashboard card.
Motivational quotes — rotates through a stored set of quotes with a refresh button.
Light/dark theme toggle, with matching logo and banner artwork per theme.
Personal settings — set your display name (shown in the banner greeting) and choose an avatar (shown in the nav bar).
Fully responsive — custom grid layouts for mobile, tablet, and desktop breakpoints.
Persistent — all data (tasks, goals, planner entries, settings, current page/tab) is saved to localStorage, so it survives a page refresh.
